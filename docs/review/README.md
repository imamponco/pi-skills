# Review Docs

Contains the review and publication workflow.

## Skills
- `code-review` → produces `review_result.md`
- `publish-message` → sends the result to Discord, Telegram, or GitLab MR

## Use when
- you want a structured diff review
- you want to send the generated review artifact to a channel

## Outputs
- `review_result.md`
- delivery status from `publish-message`
